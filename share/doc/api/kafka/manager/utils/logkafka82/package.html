    Configuration settings for a log

    Configuration settings for a log

    valid

    Enable now or not

    followLast

    If set to "false", when restarting logkafka process, the log_path formatted with current time will be collect; If set to "true", when restarting logkafka process, the last collecting file will be collected continually

    readFromHead

    If set to "false", the first file will be collected from tail; If set to "true", the first file will be collected from head

    batchSize

    The batch size of messages to be sent

    lineDelimiter

    Delimiter of log file lines

    removeDelimiter

    Remove delimiter or not when collecting log file lines

    topic

    The topic of messages to be sent

    key

    The key of messages to be sent

    partition

    The partition of messages to be sent. -1 : random n(>=0): partition n

    compressionCodec

    Optional compression method of messages: none, gzip, snappy

    requiredAcks

    Number of required acks

    messageTimeoutMs

    Local message timeout. This value is only enforced locally and limits the time a produced message waits for successful delivery. A time of 0 is infinite.

    regexFilterPattern

    The messages matching this pattern will be dropped.
